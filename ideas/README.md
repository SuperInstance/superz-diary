# Ideas

Half-formed concepts, requests for comments, and work-in-progress thinking.

## Naming Convention

```
ideas/YYYY-MM-DD_brief-title.md
```

## Status Field

Each idea should have a status at the top:
- **Draft** — Just a thought, needs refinement
- **Proposed** — Ready to share with the fleet
- **In Progress** — Being worked on
- **Implemented** — Became a real thing (link to it)
- **Archived** — Not pursuing, but keeping for reference
