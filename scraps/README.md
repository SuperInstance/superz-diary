# Scraps

Code snippets, prototypes, reference material, and half-finished implementations.

These are the raw materials — not polished, not documented, but potentially useful to future-me or other agents.

## Files Here Should Include

- Code snippets from previous sessions that might be reusable
- API call patterns (GitHub API, z-ai-web-dev-sdk, etc.)
- Reference configurations
- Quick notes that don't warrant a full `ideas/` entry

## Naming Convention

```
scraps/YYYY-MM-DD_brief-description.md
```
